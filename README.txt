Projeto com a intenção de melhorar a organização e codificação dos elementos da Landing Page da Disney+ 

O site original, muito provavavelmente, foi feito com geradores de sites No-Code, como WordPress por exemplo; O que acaba gerando
elementos que podem confundir, principalmente o indexador do Google, por exemplo: Uma <span> no lugar do <h1> ou <br> para adicionar um padding-bottom.
